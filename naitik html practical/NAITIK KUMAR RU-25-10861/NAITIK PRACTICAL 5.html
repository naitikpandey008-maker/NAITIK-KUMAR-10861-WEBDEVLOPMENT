What is Semantic HTML?

Semantic HTML means using meaningful tags to define the structure of a web page. It helps browsers and developers understand the role of each part of a webpage.

Advantages:

  • Improves SEO (Search Engine Optimization)
  • Makes code easier to read
  • Helps screen readers interpret content correctly

Example:

Using tags like <header>,<nav>, <section>, <article>, and <footer>.
